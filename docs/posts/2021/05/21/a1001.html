랭크 사용 언어

🔗 전체 1001번 문제

조건 🔗

시간제한 메모리 제한
2초 128MB

문제 🔗

두 정수 A와 B를 입력받은 다음, A - B를 출력하는 프로그램을 작성하시오.

입력 🔗

첫째 줄에 A와 B가 주어진다. (0<A,B<10)

출력 🔗

첫째 줄에 A - B를 출력한다.

케이스 🔗

  • 입력

TC

03 2
  • 출력

TC

01
